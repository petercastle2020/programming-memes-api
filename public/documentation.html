Programming Memes API Documentation

Get General Meme

Retrieves a general programming meme.

  • Endpoint: /general
  • Method: GET

Example:

GET https://programming-memes-api.onrender.com/api/meme/general

Response:

{
    "id": 1,
    "language": "general",
    "imageUrl": "http://res.cloudinary.com/dbtdebbei/image/upload/v1685556678/general-programming-memes/ttpjya9t60cpng4kfgca.webp"
  }

Get Specific Language Meme

Retrieves a programming meme specific to the given language.

  • Endpoint: /language/:language
  • Method: GET
  • Available languages: python, javascript

Example:

GET https://programming-memes-api.onrender.com/api/meme/language/javascript

Response:

{
    "id": 1,
    "language": "javascript",
    "imageUrl": "http://res.cloudinary.com/dbtdebbei/image/upload/v1685555877/javascript-memes/beihl8fgebdgixbutz4p.webp"
  }

Error Handling

If there is an error during the request, the API will respond with an appropriate error message and status code.

Possible error responses include:

  • 400 Bad Request: Invalid language parameter or missing required parameters.
  • 404 Not Found: No memes found for the specified language or general category.
  • 500 Internal Server Error: An internal server error occurred.

Please ensure you handle these errors gracefully in your client application.

Rate Limiting

To prevent abuse and ensure fair usage, the "/api/meme" endpoint has a rate limit of 50 calls per minute.

If you exceed this limit, you will receive a "429 Too Many Requests" response. Please respect the rate limit to ensure a smooth experience for all users.
